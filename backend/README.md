# UWDP - Backend

Default port: 8000

## Endpoints
- /api/auth/login/
    - Login a user (POST)
        - "username"
        - "password"

- /api/auth/register/
    - Create a new user (POST)
        - "username"
        - "password"
        - "email"

- /api/courses/
    - Add a course (POST)
        - Not implemented

    - List all courses (GET)
